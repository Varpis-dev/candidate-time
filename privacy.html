Политика конфиденциальности

Настоящая политика конфиденциальности описывает, как приложение «Время кандидата» обрабатывает данные пользователей.

1. Сбор данных

Приложение не собирает, не хранит и не передает персональные данные пользователей на сторонние серверы.

2. Использование данных

Приложение использует данные, доступные через Bitrix24 (например, город кандидата), исключительно для отображения локального времени.

3. Передача данных

Данные не передаются третьим лицам.

4. Безопасность

Приложение не хранит данные вне инфраструктуры Bitrix24.

5. Изменения

Политика может быть обновлена без предварительного уведомления.
